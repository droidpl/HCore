Member Function Documentation

void AbstractOgreNegotiator::notifyMediator ( NegotiatorEvent &  mediatorEvent)
virtual

Notifies the mediator with an event to order it an actions.

Actions sent to the mediator are only needed when multiple negotiators are involved in this action. If the negotiator can perform this action itself or the only implied negotiator is the one that sends the event, then this event might not be sent.

Parameters
mediatorEventThe event that will be sent to the mediator.
